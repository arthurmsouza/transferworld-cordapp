TransferWorld

Use this CorDapp via the node shell or the api which can be found at http://host:post/api/transferworld, end-points:
  • Send an amount of money /api/transferworld/transfer?target=NODE&amount=VALUE where NODE is the name (X500 organisation) of the node you want to transfer amount of money and VALUE amount of money!
  • List your all amounts of money: /api/transferworld/amounts
  • Get a list of peers: /api/transferworld/peers
  • Get your node's name: /api/transferworld/me
